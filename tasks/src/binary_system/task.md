# Aufgabe: Umrechnung im Binärsystem

## Aufgabenstellung

### Binärsystem → Dezimalsystem

Rechnen Sie die folgenden Binärzahlen in Dezimalzahlen um:

- $1001101_2$
- $11101001_2$
- $101000110_2$
- $110110,10011_2$
- $1000110,010011_2$

### Dezimalsystem → Binärsystem

- $143_{10}$
- $56_{10}$
- $226_{10}$
- $315,781_{10}$
- $83,8429_{10}$
